# 인프런 익스텐드
인프런에서 없는 기능을 추가할 수 있는 chrome extension 입니다.

<img src='./images/courseScreenShot.png'>

# 기능
- 영상 자동재생
- 영상 배속 증가 (5x 까지)
- 홈화면에 내강의 바로 검색기능
